# Land Type Classification using EuroSAT

## Project Overview
This project focuses on classifying land types using the EuroSAT dataset (Sentinel-2 satellite images). The goal is to accurately identify different land categories using machine learning models.

## Kaggle Links
- [Project Code](https://www.kaggle.com/code/mohamedelnahry/land-type-classification-eurosat)
- [Dataset](https://www.kaggle.com/datasets/mohamedelnahry/land-type-classification-using-eurosatsentinel-2)

## Features
- Classification of different land types
- Uses machine learning techniques for accurate predictions
- Visualization of results with plots and graphs

## Libraries & Tools
- Python
- NumPy, Pandas
- OpenCV
- PyTorch / TensorFlow (depending on the model used)
- Matplotlib / Seaborn for visualization
